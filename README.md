# clean-blue-portfolio

A personal website for a programmer, featuring a clean and modern design. The site includes sections for About Me, Portfolio, Blog, and Contact information. The color scheme is professional with shades of blue and gray, and the layout is user-friendly with intuitive navigation.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/clean-blue-portfolio.git
cd clean-blue-portfolio
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
